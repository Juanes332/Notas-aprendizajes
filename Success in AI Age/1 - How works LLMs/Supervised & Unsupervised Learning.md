# Supervised & Unsupervised Learning

## Supervised Learning

El aprendizaje supervisado se aplica cuando de manera explicita mapeamos la data entregada a la maquina, es decir la tagueamos. Por ejemplo cuando queremos hacer que nuestra maquina aprenda que es un gato, no solo le pasamos imágenes de gatos, si no que también de manera explicita le decimos que estas son imágenes de gatos, de esta manera podrá aprender por referencia y de manera **supervisada** los patrones de un dataset

## Unsupervised Learning

El aprendizaje no supervisado lo usamos cuando queremos que la propia maquina aprenda clusterizando los datos a partir de patrones. Esto lo que permite es la clasificación de los datos de manera automática según el patron que sigan, Spotify usa este tipo de aprendizaje con la clasificación de géneros musicales.
