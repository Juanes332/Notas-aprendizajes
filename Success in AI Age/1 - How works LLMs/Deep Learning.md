# Deep Learning

El deep learning se utiliza para las tareas mas complejas, este imita como funciona las conexiones neuronales de los humanos por medio de las neuronal networks (redes neuronales), estas redes neuronales conectan neuronas con otras (teniendo en cuenta las neuronas como nodos).
Cada capa de las redes neuronales están basadas en edges, shapes y emotions; esto permite que cada vez que queramos hacer que un modelo reconozca un patron debería seguir estos 3 pasos para poder dar un veredicto (ej: reconocer si en la imagen hay un gato, un perro, un loro, etc).

Esta tarea por ejemplo nos sirve demasiado cuando estamos realizando clasificaciones de imágenes, verificación biométrica, clasificación de objetos, etc.
