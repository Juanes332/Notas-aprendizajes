# Como funciona el cloud computing


Lo que todos ya conocemos es que siempre un cliente va a conectarse a un servidor y ahi pasan cosas. Pero donde esta ese servidor, que características, que servicios y capacidades tienen. 
El cloud computing es en palabras cortas la capacidad de conectarnos a servidores externos para poder realizar operaciones, un cliente siempre se pregunta a un servidor por la paquetería que necesita, ya sea una app web, un website básico o funcionalidad (login, data para dashboard, multimedia, etc).

Cuando hablamos de cloud computing hablamos de como podemos usar estos servidores para poder usar estos servicios para nuestros propósitos, plataformas como aws nos permiten "alquilar" servicios cloud (servidores) para que podamos usarlos en nuestros proyectos y aplicaciones.
La idea fundamental de esto es que tenemos como consumidores la posibilidad de usar maquinas de una empresa sin la necesidad de tener que realizar un gasto a nivel de infraestructura.

Existen varios precios para los servicios que ofrece aws, regiones y responsabilidades de uso.
Como usuarios podemos gestionar todos estos servicios y productos por una consola web que sirve como catalogo se servicios y de punto de entrada para el despliegue, uso y eliminación de los servicios que nos ofrece la plataforma cloud
