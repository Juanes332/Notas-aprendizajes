Un algoritmo es un conjunto de instrucciones para una tarea compleja, cada pieza de codigo puede llamarse un algoritmo pero existen patrones y soluciones a problemas comunes donde podemos aplicar dichos algoritmos.


## Algoritmo binary search
Este algoritmo nos permite encontrar un elemento de manera mas eficiente en una lista **ordenada** usando logaritmos (log siempre significa log2 - tenerlo en cuenta para Big O).
Binary search solo funciona en una lista ordenada, ya sea alfabética o numeral mente.
